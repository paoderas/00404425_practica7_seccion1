* ¿Cuál es la diferencia entre autenticación y autorizacion?
  Autenticacipón se refiere al proceso de revelar y verificar identidad, mientras que autorización es validar dependiendo de quién se autentica, permitir acciones dentro de la web.
* ¿Cuál es la función del token JWT en la guía?
Permite la comunicación cliente-servidor sin tener que verificar contraseña e identidad cada vez, si no que lo hace mediante un token.
